## Projects

-  [**ShebaHub**](https://github.com/hubhub20256/ShebaHub) – A medical research collaboration platform developed with **Sheba Hospital**, connecting students and interns with researchers. Built with a React frontend and robust backend APIs. - [**The website**](https://shebahub.hitheal.org.il/)
-  [**JumpsCare**](https://github.com/AlonDlv/JumpScare) – Chrome extension that automatically warns of jumpscares on Netflix, powered by a fully serverless AWS backend utilizing Lambda, DynamoDB, and API Gateway. - [**Chrome Web Store**](https://chromewebstore.google.com/detail/gendljahhmmnaoillkkkbibmildoeoca?utm_source=item-share-cb)
-  [**AcademyLink**](https://github.com/AlonDlv/AcademyLink) – Campus accessibility and awareness project built for **Academy Link HIT** in collaboration with **Link20**.
-  **Cost Manager** – A RESTful application developed for Client-Side and Server-Side Development courses.
